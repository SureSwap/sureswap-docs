# SureSwap Unified Wallet

**SureSwap** is a multichain self-custody wallet that lives inside Telegram. One account, one set of keys, and addresses across every major network — no app downloads, no browser extensions, no seed phrases to copy.

---

## What you can do

| Feature | Description |
|---------|-------------|
| 💸 **Send & Receive** | Transfer tokens on EVM chains, Solana, TON, and Tron directly in Telegram |
| 🔄 **Swap** | Same-chain and cross-chain swaps powered by Jupiter, 0x, and aggregated DEX routing |
| 🌉 **Bridge** | Move assets between networks via Mayan, Relay, and LayerZero |
| 📉 **Limit Orders** | Set buy/sell orders that execute automatically when your target price is hit |
| 🔒 **Privacy Mode** | Swap through a shared liquidity pool — break the on-chain link between your funds |
| 📈 **Perps Trading** | Trade perpetuals on Hyperliquid directly from your Telegram wallet |
| 🏷️ **Profiles** | Manage up to 10 separate wallet profiles from one Telegram account |
| 🔗 **WalletConnect** | *(Coming soon)* Connect your SureSwap wallet to any dApp |

---

## Supported networks

SureSwap supports **Solana, Ethereum, Base, BSC, Arbitrum, HyperEVM, Plasma, Abstract**, plus **TON** and **Tron** for send/receive. See the full [Supported Networks](./supported-networks/) page for details.

---

## Getting started

New to SureSwap? Start here:

1. Open the bot in Telegram and tap **Create Unified Wallet**
2. Your EVM, Solana, TON, and Tron addresses are created instantly — all backed by the same keys
3. Deposit some tokens and start trading
