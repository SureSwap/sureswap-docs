# WalletConnect

{: .coming_soon }
WalletConnect support is coming soon. You'll be able to connect your SureSwap wallet to any WalletConnect-compatible dApp — DeFi protocols, NFT marketplaces, and more — directly from Telegram.

When available, the flow will work as follows:

1. Open the dApp in your browser and click "Connect Wallet" → "WalletConnect"
2. Copy the WalletConnect URI or scan the QR code
3. In SureSwap, tap **WalletConnect** and paste the URI
4. Approve the connection
5. Sign transactions from the dApp directly inside Telegram — no browser extension needed
